# Credit Risk Analysis Project

## 📌 Objective
To identify high-risk loan applicants using financial and demographic data to support better lending decisions.

## 🛠 Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Python (Modeling - Logistic Regression)
- Power BI (Dashboard)

## 🔍 Approach
- Cleaned and transformed data in Excel
- Performed risk segmentation using SQL
- Built a logistic regression model in Python
- Created an interactive dashboard in Power BI

## 📊 Key Insights
- High-risk customers have significantly higher loan-to-income ratios
- Lower income groups show higher risk concentration
- Financial stress is a major indicator of risk
